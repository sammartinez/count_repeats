
-------------Zero Case----------------
Behavior: a (single-letter word) will show up once  

Input 1: "a"
Input 2: "a"

Output: 1
-------------First Case----------------
Behavior: a (single-letter word) will show up twice

Input 1: "a"
Input 2: "a unicorn can only eat a cup of food"

Output: 2
-------------Second Case----------------
Behavior: bear (multiple-letter word) will show up once

Input 1: "bear"
Input 2: "bear is a dog that sam owns"

Output: 1
-------------Third Case----------------
Behavior: tacos (multiple-letter word) will show up twice

Input 1: "tacos"
Input 2: "i love tacos tacos are a way of life"

Output: 2
-------------Fourth Case----------------
Behavior: converts every input into lowercase

Input 1: "Taco"
Input 2: "i love tacos tacos are a way of life"

Output: 2
-------------Fifth Case----------------
Behavior: Replaces special characters with spaces

Input 1: "a"
Input 2: "a dog.. a dog is a friend that loves you, well sometimes."

Output: 3

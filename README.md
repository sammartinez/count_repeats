# Count Repeats

##### Counts how many times a word is within a sentence, 08/14/15_

#### By _**Sam Martinez**_

## Description

This application takes a word inputted by a user and displays how many times the word appears within a sentence.

## Setup

* Clone Repository from github
* Run composer install in the project folder
* Start php server in web folder on the project folder : php -S localhost:8000
* In a web browser, navigate to the localhost:8000 and enjoy!


## Technologies Used

- HTML
- CSS Bootstrap
- PHP
- Silex
- Twig
- PHPUnit
